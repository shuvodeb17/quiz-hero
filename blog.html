1. What is the difference between map(), filter(), forEach(), find() ?

Ans: map() loops over each element in an array. It creates a new array. Does not modify the original array. filter() is used on arrays. Given a condition in filter , it checks each element of the array. Creates a new array that satisfies the condition. forEach() is used to check each element of an array. It does not change the array. It returns undefined. Given a condition in find(), if multiple elements of an array match, it will return only the first element as an object.

2. What is the difference between localStorage, sessionStorage, cookie?

Ans: By storing the data in localStorage, if the user deletes the browser tab and visits the website again, he can access it. If the data is stored in sessionStorage then all the data will be lost if the user deletes the tab. It improves website performance. Cookies are a good choice for storing data that should not be retained for long periods of time.

3. What is the event loop and how does it work?

Ans: The event loop works by making a request to some internal or external "event provider" (that generally blocks the request until an event has arrived), then calls the relevant event handler.
